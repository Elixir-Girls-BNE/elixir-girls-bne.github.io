Welcome to Elixir Girls!

These guides have been developed by our fantastic community of volunteer mentors, some of whom you will meet today. These guides are designed to make learning to develop with Elixir and Phoenix more approachable and to help build a diverse and inclusive community. Get started by choosing the install guide for your operating system either Mac, Windows or Linux

If you are looking for a place to start, try our Beginner's Guide!

If you would like to contribute to these guides you can do so by forking the repository and creating a pull request on Github
