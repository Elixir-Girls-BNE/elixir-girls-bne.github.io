Setting up Our App

The first thing that we want to do is to create our Phoenix app we do this with the following command and we pass in slackir as the name of our app.

$ mix phoenix.new slackir

The next thing we want to do is move into our newly created application to do that we run

$ cd slackir

Now we can install our applications dependencies by running

$ npm install

and we can create our database by running

$ mix ecto.create

To start the server you will need to run

$ mix phoenix.server

and go to http://127.0.0.1/4000 to view your new application
